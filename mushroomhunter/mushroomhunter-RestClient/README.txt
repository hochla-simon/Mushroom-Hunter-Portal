Mushroom-Hunter-Portal
======================

PA165 Team Project - Mushroom Hunter Portal - REST Client

Project Description
---------------------
This project contain REST Client - Swing desktop application developed as 
school team project. This application is intended for testing of REST API of 
Mushroom Hunter Portal web application.

Prerequisites 
--------------------
1)  Java 7 installed
2)  Maven 3 installed
3)  Path to Maven and Java set in environment variables
4)  Mushroom Hunter web application running on same machine 
    (available on http://localhost:8080/pa165)
     
How to run project
--------------------
1)  Open command line and go to REST client project
    (REPOSITORY_BASE/mushroomhunter-RestClient)
2)  Execute command:
    mvn exec:java
